# Qwen3-TTS-test
Qwen3 TTS test codes.

## ファイル説明

### `check.py`
`Qwen3TTSModel` を使用した音声生成の実行時間を計測するスクリプトです。
直接音声を生成する場合と、事前にボイスクローンプロンプトを作成して生成する場合のパフォーマンスを比較します。

### `make_wav.py`
ユーザー入力から音声ファイルを生成するための対話型スクリプトです。
"end" が入力されるまで、指示 (instruct) とテキストの入力を受け付け、WAVファイルを生成し続けます。

### `test01.py`
`generate_custom_voice` を使用して英語の音声を生成するシンプルなテストスクリプトです。
話者 "Ryan" を使用します。

### `test02.py`
`generate_custom_voice` を使用して日本語の音声を生成するシンプルなテストスクリプトです。
話者 "Ono_Anna" を使用します。

### `test03.py`
参照音声ファイルを使用してボイスクローンを行う `generate_voice_clone` のテストスクリプトです。
`instruct` パラメータ ("As the news anchor says") を含んでいます。

### `test04.py`
ボイスクローンプロンプトを事前に計算してボイスクローンを行うテストスクリプトです。
より長い日本語テキストに対し、口調や速度に関する詳細な指示を与えて音声を生成します。
